Html

  1. Name the components of html elements properly
    2. The components are text header using <h1> to <h6>, a paragraph using <p>, a link using <a>, a list using <ul> or <ol> and <li> for listing its list elements, an image using <img>
  2. Identify void elements and how they are denoted
    3. They only have a start tag since they can't have any content
  3. Identify attributes
    4. - All HTML elements can have attributes
    - Attributes provide additional information about an element
    - Attributes are always specified in the start tag
    - Attributes usually come in name/value pairs like: name="value"
  4. Write down the correct DOCTYPE declaration for html 5
    5. <!DOCTYPE html>
  5. Write down syntactically correct statements for the following elements
    • Article (<article>): <article><h1>sample</h1></article>
    • Body (<body>): <body><h1>sample</h1></body>
    • line break (<br/>): <p>sample<br>sample</p>
    • Headings (<h1>, ...): <h1>sample</h1>
    • Section with meta-information (<head>): <head><title>This is a title</title></head>
    • Top level element (<html>): <html><head></head></html>
    • Image named Team.jpg (<img src="Team.jpg">): <img src="Team.jpg">
    • Paragraph (<p>): <p>sample</p>
    • Section (<section>): <section><h1>sample</h1></section>
    • Label appearing in the title bar (<title>): <title>sample</title>
    • Link to other pages (<a>): <a href="https://www.google.at">
    • Comment: <!-- comment -->
    • Head with title and meta elements: <head><title>sample</title><meta charset= "utf-8"></style></head>

Css

  1. Name the components of a css rule properly
    2. selector { property: value;}
  2. Use combinators properly: direct child, descendant, adjacent sibling, sibling
    3. child selector (>): selects all elements that are the children of a specified element
    descendant selector (space): selects all elements that are descendants of a specified element
    adjacent sibling selector (+): selects all elements that are the adjacent siblings of a specified element
    general sibling selector (~): selects all elements that are siblings of a specified element
  3. Specify colors by color names and in hexadecimal notation
    4. blue #0000ff
    red #ff0000
    green #00ff00
  4. Declarations for color, background color
    5. background-color: blue;
    color: blue;
  5. Pseudo classes for the <a> element
    6. a:active
    a:hover
    a:link
    a:visited
  6. Declaration for background image
    7. background-image: url("sample.png");
  7. Declaration for text alignment (left, right, center, justify)
    8. text-align: left;
    text-align: right;
    text-align: center;
    text-align: justify;
  8. Declaration for text decoration (overline, underline, line-through)
    9. text-decoration: overline;
    text-decoration: underline;
    text-decoration: line-through;
  9. Declaration for text transformation (uppercase, lowercase, capitalize)
    10. text-transform: uppercase;
    text-transform: lowe rcase;
    text-transform: capitalize;
  10. Identify the difference between serif and sans-serif fonts
    11. sans-serif means sans “without” strokes
  11. Declarations for font families
    12. font-family: Arial;
  12. Declarations for font style normal and italic
    13. font-style: normal;
    font-style: italic;
  13. Declarations for font sizes
    14. font-size: 20px;
  14. Declarations for font weights
    15. font-weight: 300;
